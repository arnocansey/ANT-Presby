# ANT PRESS Frontend

Next.js frontend for the ANT PRESS workspace.

## Commands

```bash
npm run dev
npm run build
npm run start
npm run type-check
```

## Notes

- `/admin` redirects to `/admin/dashboard`.
- Admin create/edit flows for sermons and events were aligned with backend payloads.
- Visible app branding now reads from `NEXT_PUBLIC_APP_NAME` with `ANT PRESS` as fallback.
